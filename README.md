# AI Career Curriculum - 장효준 독립 포트폴리오 사이트

이 프로젝트는 기존 `aimirae` 홈페이지와 분리해서 운영하는 독립 GitHub 저장소용 MPA 정적 웹사이트입니다.

## 권장 로컬 경로

```text
C:\Users\안지은\Documents\GitHub\ai-career-curriculum
```

## URL 구조

```text
/                                      메인: AI 강사 장효준 소개
/classes/ai-basic/                    AI 기초 클래스 입문반
/classes/ai-ad-marketing/             AI 광고제작 마케팅 전문가 클래스
/classes/ai-video-creator/            AI 영상 크리에이터 전문가 클래스
/classes/ai-workflow/                 AI 실무자동화 워크플로우 과정
```

## GitHub Desktop 업로드 순서

1. `C:\Users\안지은\Documents\GitHub` 아래에 `ai-career-curriculum` 폴더를 둡니다.
2. GitHub Desktop 실행
3. `File` → `Add local repository` → `ai-career-curriculum` 선택
4. 좌측 변경사항 확인
5. Summary에 `initial ai career curriculum site` 입력
6. `Commit to main`
7. `Publish repository`
8. GitHub 웹에서 저장소 `Settings` → `Pages`
9. `Deploy from a branch` → `main` → `/root` → `Save`
