# AI Career Curriculum

AI 강사 장효준의 커리어 소개와 과정별 커리큘럼을 독립 저장소로 운영하기 위한 MPA 방식 정적 웹사이트입니다.

## 구조

```text
ai-career-curriculum
├─ index.html
├─ classes
│  ├─ ai-basic/index.html
│  ├─ ai-ad-marketing/index.html
│  ├─ ai-video-creator/index.html
│  └─ ai-workflow/index.html
├─ assets
│  ├─ css/style.css
│  ├─ js/main.js
│  └─ images/jang-hyojun-profile.webp
└─ .nojekyll
```

## 포함된 효과

- 대표사진 기반 메인 프로필 카드
- AI 데이터가 흐르는 전자 회로형 배경 애니메이션
- 마우스 클릭 시 푸른 번쩍임 효과
- 내부 페이지 이동 시 블랙홀/웜홀 전환 효과
- 각 과정별 독립 HTML 페이지 구성

## GitHub Desktop 사용 흐름

1. `C:\Users\안지은\Documents\GitHub\ai-career-curriculum` 폴더에 압축 해제
2. GitHub Desktop에서 `File > Add local repository`
3. Git 저장소가 아니라고 뜨면 `create a repository here` 클릭
4. `Commit to main`
5. `Publish repository`
6. GitHub 저장소 `Settings > Pages`에서 `Deploy from a branch`, `main`, `/root` 선택

예상 배포 주소:

